# More HTML Elements

You know `<h1>`. Here are four more tags you'll use constantly.

## `<p>` — Paragraph

```html
<p>This is a paragraph of text.</p>
```

## `<h2>` — Subheading

Headings go from `<h1>` (biggest) to `<h6>` (smallest). Use `<h2>` for section titles.

```html
<h2>About Me</h2>
```

## `<a>` — Link

The `href` attribute tells the browser where the link goes.

```html
<a href="https://google.com">Go to Google</a>
```

## `<img>` — Image

Images are self-closing — no `</img>` needed. `alt` is the fallback text.

```html
<img src="photo.jpg" alt="A description of the photo" />
```

Now build a small personal page using all four tags.
