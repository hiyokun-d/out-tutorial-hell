# What is Code?

Code is just **instructions** written in a language a computer can understand.

That's it. Nothing magical. When you write code, you're telling the computer exactly what to do, step by step.

## Why do websites exist?

Every website you've ever visited — Google, YouTube, Reddit — is built with code. Specifically three languages:

- **HTML** — the structure (what's on the page)
- **CSS** — the style (how it looks)
- **JavaScript** — the behavior (what it does)

Think of it like building a house:
- HTML is the walls and rooms
- CSS is the paint and furniture
- JavaScript is the electricity and plumbing

## You already know more than you think

You've used websites your whole life. You know what a button is, what a form is, what a link does. That's the mental model. Now you're going to learn how to *build* those things.
