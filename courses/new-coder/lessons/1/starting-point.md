# What is programming?

A program is a set of instructions you give to a computer. The computer follows them exactly — no more, no less.

Think of a recipe. A recipe tells a cook: do this, then this, then this. A program does the same thing for a computer.

## Computers are literal

This is the most important thing to understand as a new coder:

**Computers do exactly what you say, not what you mean.**

If you say "add some salt", a cook knows what that means. A computer doesn't. You'd have to say "add 5 grams of salt". Precise. Unambiguous.

This is why programming languages exist — they give you a way to write precise instructions.

## The console

Programs often need to show output — something they calculated, a message, a result. One of the simplest ways to do that in JavaScript is `console.log()`.

```javascript
console.log("Hello!")
```

This tells the computer: print the text `Hello!` to the output.
